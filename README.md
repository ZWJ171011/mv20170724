### 分支

- 新建分支
    > grt branch 分支名

- 查看分支
    > git branch

- 切换分支
    > git checkout 分支名

- 快速创建 + 切换
    > git checkout -b 分支名

- 合并分支
    > git merged 分支名

    可能会出现的冲突
        人为修改文件 -> 提交

- 删除已合并的分支
    > git branch -d 分支名

- 删除未合并的分支
    > git branch -D 分支名

- 查看已合并的分支
    > git branch --merged

- 查看未合并的分支
    > git branch --no-merged
